キカガクは「目指したくない教育者」を再定義し、
教育者を目指したい職業にアップデートしていきたいと考えています。
雇用条件等の待遇の改善、新しい知識をアップデートできる環境を整備、実務に沿った学びを提供する等、
「本当の教育者とはこうあるべき」ということをキカガクでは教育業界の先駆者として体現していきます。
そして、あるべき教育によって育った優秀な人材を投資し社会に還元していきます。
人に教え、人を育み、その過程で培った知識を還元していく。
これが私たちの目指す教育です。

雇用条件
講師希望の方は、試用期間（契約社員）として最初の3ヶ月間に研修を用意しております。
試用期間では、AIに関する知識を習得していただくだけでなく、講義の練習を実施していただき、研修開始2ヶ月後に行われる入社テスト(模擬授業)にて合格すれば、正社員登用として新たな雇用契約を締結いたします。
試用期間(契約社員)

  • 福利厚生：社会保険完備、家賃補助3万円/月
